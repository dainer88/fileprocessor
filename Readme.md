### Readme.


## Requerimientos.

Un servicio de MongoDB corriendo.

Python 2.7.6.

Django 1.11.2.

Pymongo 3.7.1. Utilizar MongoDB con Python.

Mongoengine. Capa ORM para trabajar encima de Pymongo.


## Visualización.

Para visualizar el proyecto a través del servidor de desarrollo de Django. En la raíz del proyecto (al nivel de manage.py) ejecutar.

python manage.py runserver

En el navegador ir a.

http://localhost:8000/crud
